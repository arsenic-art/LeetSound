LeetSound – Chrome Extension

🚀 Installation : 
***Open Chrome Extensions Page***
In Chrome, go to: ***chrome://extensions/***


**Then,** Enable Developer Mode
**Toggle Developer Mode** (top-right corner)


**Load the Extension :** 

Click Load unpacked
Select the extracted project folder
Click Select Folder


AAAND Booooomm that's it you are good to go, enjoy!!!


🧪 How To Use

Go to https://leetcode.com
Submit any solution
When the result appears, a sound will play automatically
