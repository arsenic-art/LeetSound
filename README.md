# 🔊 LeetSound – Chrome Extension

LeetSound is a fun Chrome extension that plays sound effects based on your **LeetCode submission results**.

No more silent Accepted.
No more boring Wrong Answers.
Now your submissions have personality 🎧🔥

---

## 🚀 Installation

Since this extension is not published on the Chrome Web Store, you need to load it manually.

### Step 1: Open Chrome Extensions Page

Open Chrome or Chromium based browsers and go to:

chrome://extensions/

---

### Step 2: Enable Developer Mode

Toggle **Developer Mode** (top-right corner).

---

### Step 3: Load the Extension

- Click **Load unpacked**
- Select the extracted project folder
- Click **Select Folder**

AAAND Booooomm 💥  
That's it — you're good to go. Enjoy!!!

---

## 🧪 How To Use

1. Go to https://leetcode.com  
2. Submit any solution  
3. When the result appears, a sound will play automatically  

Make sure your system volume is ON 🔊
